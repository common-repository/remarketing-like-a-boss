== Description ==
Easily add Remarketing Code to your website

Add ReMarketing Code to every page in your website easily using this 1x simple plugin. 

Retargeting plaforms - Google, Facebook, Retargetter, Adroll, Perfect Audience, Bing and also as a bonus can add Google Analytics code also. 

Allows different Remarketing Code on individual Page/Post if you just want to use different campaign for that page/post.

For more information, visit; <a href="http://webhaven.com/web-design-brisbane/">Website Design Brisbane</a>

== Installation ==

1. Download Remarketing Like a Boss!
2. Upload the Remarketing Like a Boss! directory to your /wp-content/plugins/ directory. Using your favourite method (ftp, sftp, scp, etc)
3. Activate Remarketing Like a Boss from your Plugins page. (It will appear in the left hand column)
